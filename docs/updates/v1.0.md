# New in Version 1.0

Cache Cleaner has been upgrade to version 1.0.

The extension still works the same way but has a slightly different look and feel with some new options added.

[[toc]]

### Changes

- The [popup](../usage.md#toolbar-popup) now shows all buttons. Can be reverted in [options](../options.md#show-all-buttons).
- Deprecated cache options are now hidden. Can be reverted in [options](../options.md#show-deprecated-options).
- The options page shows a random background. Can be changed/disabled in [options](../options.md#options-page-background).
- The extension is now [localized](../install.md#languages) and may display in your browser's language.

### Features

- Show all flush buttons in the [popup](../usage.md#toolbar-popup).
- Set [popup](../usage.md#toolbar-popup) default view to browser.
- New [side panel](../usage.md#side-panel) UI to flush cache.

### Technical

- Complete rewrite in TypeScript and Vue3 using the [WXT framework](https://github.com/wxt-dev/wxt).
- To view a complete changelog, see the [release notes](https://github.com/cssnr/cache-cleaner/releases/tag/1.0.0) on GitHub.

For more details, see the [source code](https://github.com/cssnr/cache-cleaner) or checkout the [CONTRIBUTING.md](https://github.com/cssnr/cache-cleaner/blob/master/CONTRIBUTING.md).
